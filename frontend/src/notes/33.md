# Big Numbers!

In Chinese, you group by 4 zeros and not 3 (1,0000 not 10,000)

| 1        | 一 (yī)    |
| 10       | 十 (shí)   |
| 100      | 百 (bǎi)   |
| 1,000    | 千 (qiān)  |
| 10,000   | 万 (wàn)   |
| 100,000,000 | 亿 (yì) |

examples:
* 45 → 四十五
* 143 → 一百四十三
* 1435 → 一千四百三十五
* 10,520 → 10,520：一万零五百二十
* 12,300 → 12,300：一万两千三百
* 20,000 → 20,000：两万
* 35,700 → 35,700：三万五千七百
* 98,750 → 98,750：九万八千七百五十
* 100,000 → 100,000：十万
* 125,000 → 125,000：十二万五千
* 553,700 → 553,700：五十五万三千七百
* 1,000,000 → 1,000,000：一百万
* 1,478,943 → 1,478,943：一百四十七万八千九百四十三
* 10,500,000 → 10,500,000：一千零五十万
* 90,000,000 → 90,000,000：九千万
* 99,999,999 → 99,999,999：九千九百九十九万九千九百九十九
