# 谁:哪儿,都

S + Q.W. + 都/也 + (不/没) + V

## ALL OR NONE : 哪儿

哪儿/哪里S + 哪儿 + 都/也 + (不/没) + V (过)

- 他哪儿都去过了 : he's been everywhere
- 我哪儿都没去过 : I've been nowhere
- 哪儿都可以 : wherever is okay

## ALL OR NONE : 谁

S + 谁 + 都/也 + (不/没) + V

OR

S + 谁 + 都/也 + (不/没) + V + O

- 他谁都喜欢 : he likes everybody
- 你谁也不认识 : I don't know anybody
- 谁都(不)喜欢他 : everyone likes him
