# 的，地，得
de, de, de

1. 的
   * possessive relationship → Noun 的 Noun
   * descriptive relationship → Description 的 Noun

2. 地
   * adv + 地 + v (how someone performs the action)
   → 高兴地地说
   * main action is at the end of the phrase

3. 得
   * v + 得 + adj (general evaluation of how well someone did the action)
   → 说得很好
   * sometimes need to split
     他开车开得很快
   * v + 得 + result complement → indicate whether result can be achieved or not
   * negative version: v + 不 + result
   * v/adj + 得 + result
   → 我热得睡不着
